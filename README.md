# Calculator App

This is a calculator app made with React,js.

## Dependencies

The dependencies required are as follows:

"react": "^16.10.2",

"react-dom": "^16.10.2",

"react-scripts": "3.2.0"

To install this, please fork and clone this repo, and run the command `npm install`

## To View

Please run `npm start`.
